　　... 都看出感情来了。

　　

　　折原临也暗想，不能正面击破，就从侧面进攻。

　　

　　要是，被爆出黑西装男神是一位暴力狂的话 ...

　　

　　... 受欢迎度应该会直线下降。

　　

　　每天看着论坛上这么多迷妹花痴自己的男朋友，他真的挺介意的。

　　

　　是的，非常介意 ... 他必须承认。

　　

　　从口袋里拿出手机，他给静雄发了一条邮件。

　　

　　『明天晚上出来吃寿司吧』

　　

　　『好 - - - from小静』

　　

　　『记得穿黑西装』

　　

　　『为什么 - - - from小静』

　　

　　『没为什么，你穿来就是了』



　　

　　— — —



　　

　　到了约定的时间，折原临也躲在寿司店旁边的小巷里，看着平和岛静雄站在店门外左顾右盼。

　　

　　按照约定，男人果然穿来了西装。

　　

　　远远望去，真是帅气极了。

　　

　　金色的头发、黑色的西装、高挑的身材、英俊的脸庞 ... ...

　　

　　... 落日的余晖照射在他的身上，昏黄的光晕将他的周身笼罩起来。

　　

　　美得像一幅油画。

　　

　　折原临也觉得头脑一热，鼻血快要流出来了。

　　

　　他看了看身后被雇佣来的小混混，说道。

　　

　　「跟你说，就是寿司店门口的那个男人。你走过去，然后 ……」

　　

　　小混混点了点头，有些害怕地走出了小巷，向平和岛静雄走去，准备挑衅他。

　　

　　像嗅到了什么气味似的，平和岛静雄突然把头转向了小巷的方向，本能地提腿跑了起来。

　　

　　糟了！计划还没实施就宣告失败 ...

　　

　　这么多年，小静的鼻子怎么还是那么灵 ...

　　

　　... 临也在心里无奈地吐槽着。

　　

　　他向小巷的深处跑去，静雄紧随其后，也跑进了小巷。

　　

　　一个在前面跑，一个在后面追。

　　

　　跑着，跑着 ... 折原临也突然想到一个严肃的问题 ...

　　

　　为什么要跑？

　　

　　不是约好来吃饭的吗？

　　

　　既然计划没有败露，根本没有逃跑的理由吧 ...

　　

　　... ...

　　

　　他突然停住了脚步，身后的静雄差点撞了上来。

　　

　　「临也 ... 你跑什么啊」

　　

　　折原临也转过头，不好意思地笑了起来，像做了亏心事的孩子。

　　

　　平和岛静雄穿着黑西装的模样，由照片的尺寸，放大到真人的比例 ...

　　

　　... 清晰地出现在折原临也的面前。

　　

　　小静 ... 你你你 ... 真是太有心机了！

　　

　　... 能不能考虑一下别人的感受！

　　

　　折原临也， BOOM！

　　

　　鼻血流尽。

　　

　　卒。



　　

　　— — —



　　

　　那天晚上。

　　

　　他们，面对面坐着，吃饭。

　　

　　平和岛静雄感觉到，折原临也灼热的目光直直地停留在自己的身上 ...

　　

　　... 仿佛要把人瞪穿似的。

　　

　　他摸了摸临也的头。

　　

　　「没发烧啊 ... 你今天是怎么了 ... 眼神好奇怪」

　　

　　折原临也正了正身姿，一本正经地说道。

　　

　　「有什么了不起的啊」

　　

　　「不就是穿西装比我好看嘛」

　　

　　「小静也就这一点，能稍微比我强那么一点儿 ~ 」

　　

　　「哼！」



　　

　　— — —



　　

　　折原舞流将事情的经过原原本本地复述给平和岛静雄。

　　

　　之后。

　　

　　「临也，我今天穿了白色的西装」

　　

　　「临也，我今天穿了灰色的西装」

　　

　　「临也，我今天穿了蓝色的西装」

　　

　　「... ...」

　　

　　... 你是彩虹吗....



　　

　　— — —



　　

　　多年之后。

　　

　　折原临也的办公桌上依然放着当年平和岛静雄穿着黑西装的照片。

　　

　　每次静雄看到，总会调侃地说道。

　　

　　「临也，我明天穿西装给你看吧」

　　

　　「不！小静 ... 求你千万别再穿了 ... 我都快看吐了 ...」

　　

　　「那你留着那照片干嘛，还放在这么显眼的位置」

　　

　　折原临也看着平和岛静雄，脸上带着类似当年痴汉的表情，说道。

　　

　　「秘密。」



　　

　　FIN.

　　

　　2016/08/07  橙。
