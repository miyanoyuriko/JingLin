【 DRRR 】自寻死路 ( 2016.02.14 情人节贺文 )

　　　
  　　
  　　
  　　
  　　
　　
　　# 情人节贺文 #
　　# 就当作是静临同亽居二十题那个系列的他们吧 #
　　# 已交往前提 #
　　# 基本是汪酱被欺负了所以反欺负喵酱的小故事 #



  　　
　　
　　# # # 
　　
　　
　　
　　
　　
　　
　　
　　再过一小时，情人节，这个属于全天下恋人互诉心意的节日就要结束了。
　　
　　盯着即将午夜的时钟，平和岛静雄纳闷着交往至今每年都照惯例送上不同牌子终归一句全是昂贵巧克力的恋人，为什么今年没有半点表示。
　　
　　该不是还在为昨天的吵嘴闹别扭吧？
　　
　　那样一想，便觉得折原临也这个男人真是够小心眼的了。
　　
　　不过，当他去洗了趟澡，从浴亽室走出来的时候，客厅中央原本还空空如也的茶几上居然多了盒巧克力。
　　
　　走过去拿起来仔细端详，居然又是盒没有吃过的高级巧克力，还是特浓牛奶口味的。
　　
　　难道……难道这是临也他……
　　
　　心里又惊又喜地想着，完全忘记了十分钟前对对方的埋怨，静雄此刻心情还是颇感动的。
　　
　　虽然不知道送这盒巧克力的当事人跑哪儿去了，但想尽量把握情人节尾声好好品味巧克力滋味的静雄，撕开纸盒，拿出封着塑料模的巧克力，准备大快朵颐。
　　
　　就在这个当下——风速咻一声刮过手边。意会过来时，巧克力不见踪影了。
　　
　　不知从哪儿冒出来的临也夺走静雄刚拆开的巧克力，然后笑着说：“让你误会了真是不好意思啊。”
　　
　　“搞什么鬼，不是要给我的么！”
　　
　　“就说了嘛──让你误会了，真是不、好、意、思、吶。”
　　
　　重复了一次刚才的话不说，还在“不好意思”四字上面刻意压重音，眼前的男人表明是要惹自己不快了。
　　
　　“这个啊，不是给小静的。是我买来要给自己吃的。”
　　
　　“哈？你不是根本不吃巧克力的么！？”
　　
　　“哎呀谁知道呢，今天路上一直看到巧克力啊巧克力，送礼也好贩售也好手制也好，行销洗亽脑策略攻击下也就突然变得想吃了嘛。那么，我就开动啦。”
　　
　　那样毫不客气地说着，临也撕开了塑料封膜，当着静雄的面前大口大口地咬起巧克力。
　　
　　边咬进嘴里还边发出“唔，这奶香味虽浓又不会过分抢过巧克力的风味，可说是搭配得相当平衡的味道啊。不愧是○牌的巧克力，甜而不腻呐。”这种旁人看得到却又吃不到的可恨赞叹。
　　
　　咬牙握拳瞪着对方恶行的静雄，心里又开始埋怨起这个都要奔三的混帐男人还像个小鬼头一样耍着幼稚的挑衅手段来激怒自己。
　　
　　可恶。可恶。可恶。这混帐跳蚤实在太可恶了。
　　
　　但他没料到更恶劣的行径还在后头。临也不但津津有味地吃着，竟然还在吃到了剩下一口的时候，突然伸出手里的巧克力并递给了他。
　　
　　“这是干么。”
　　
　　“小静，看你眼巴巴地一付也想吃的样子，剩一口的份，我就大发慈悲地赏给你吧。”
　　
　　那半是嘲弄半是怜悯的眼神终于让静雄按捺不住胸口的气愤，用力挥开巧克力，直扑上去就把临也压倒在地。
　　
　　事出突然，以为要被揍了，临也下意识地抬起手臂护住脸颊。可他没料到静雄竟是将那股怒气化作咬食的冲动，掐住了毫无防备的颈子就是狠狠撕咬了一口。
　　
　　“啊──！！”
　　
　　临也紧闭双眼，吃痛地溢出眼泪。
　　
　　“你这野兽……疼、”
　　
　　他的话没能说完，静雄趁着他把护着脸的手伸去捂住肩颈那个瘀血伤口的空隙时，抬起他的下巴就猛吻下去。
　　
　　“啊、唔……”
　　
　　就因为措手不及，连牙齿都来不及闭合来阻挡对手的攻击，临也整个口腔内侧都被静雄粗暴地用舌头舔亽了个遍。
　　
　　一如静雄预料的，临也刚才吃得很急，还有很多巧克力残块留在嘴里未吞咽下肚。他一边享受着与对方深吻的快亽感，一边品尝着临也舌尖上渐融的巧克力香味。他又吸又吞，一付要将对方从舌头开始拆吃入腹的气势。
　　
　　临也被长达数分钟如此暴烈地强吻着，难以换气，脖子又疼。唾液顺着颊边断断续续流下来，但根本没有余欲擦掉。看起来十分难受。从临也的眼里流露的神色能隐约察觉到他正在害怕，颤动的喉头不断发出呜咽的声音。
　　
　　那半是挣扎，半是讨饶的模样，似乎是希望自己能就此停手。但他会这么快放软身姿，想必也是预测到静雄目前的打算了。既然明白，就不用浪费无用工求饶了。
　　
　　现在的静雄，对临也手中那一小口巧克力完全不屑一顾。
　　
　　他要享受的，是刚狩猎到手的更大更好的猎物。
　　
　　他要狠狠舔亽弄临也，直到那家伙哀嚎着求自己停下却也完全不放过他的程度。
　　
　　谁让他居然敢这样挑衅自己。
　　
　　这样勾引自己。
　　
　　找死。
　　
　　
　　
　　
　　
　　
　　
　　- The End -
　　
　　
　　
　　后记：
　　
　　
　　
　　抱歉，又是这种霸道不讲理的静雄与拼命作死的临也系列了……不过久违的写了两人Kiss的福利，希望众客倌满意。
　　
　　感谢阅读。情人节快乐。
