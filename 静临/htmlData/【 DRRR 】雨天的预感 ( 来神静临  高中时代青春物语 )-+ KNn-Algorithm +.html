【 DRRR 】雨天的预感 ( 来神静临 / 高中时代青春物语 )

　　# 萌得不要不要的来神静临 #
　　# 两个高中男孩子蠢萌的青春物语 #

　　最近一定被诅咒了。
　　一定是的。
　　不然为什么会经常无端遭遇水难呢？
　　他站在停放自行车的雨棚径自想道。
　　
　　
　　前天才刚跟小静在泳池边上打架，谁知一个不慎，踩空落进池子里，喝了不少带有消毒水味儿的水，回想起来就犯恶心。
　　这会儿才换好鞋，刚步出教学楼，随即，视线范围内天空满布乌云，伴随远处阵阵雷声，雨点冷不丁地直从头顶滴答滴答落下。
　　稀落的雨点声还逐进加紧，脚边干燥的地面不久打得湿漉漉的，坑洞里积起了大小不一的水洼。
　　啊啊，早上出门前，还叮嘱过家里两个丫头带上雨衣，反观自己却忘记带伞了……失策。
　　
　　
　　“临也君哟──你这混账，又把我的鞋子扔到外头来淋雨了，想死吗！！”
　　
　　思绪被打断了，折原临也扭头往教学楼的方向看去，叹了口气。
　　
　　──哎哎，那个吵人的大怪兽又在扯着嗓子鬼吼鬼叫了。
　　──小静怎么就不感谢我帮他预先把鞋子给拿到柜子外呢？就会大惊小怪的。
　　
　　看着穿了每踩一脚都能溢出雨水的鞋子，却仍然直朝自己飞奔而来的身影，那笨拙的模样，怎么看都觉得很好笑。
　　
　　“抓住你这家伙了！！”平和岛静雄一边说着，还一边“嘿嘿”笑得很得意。那笑容怎么看都觉得狰狞。不知道碰过什么而显得脏兮兮的手，正揪紧了自己左边的衣襟。
　　
　　──是说，就算我身上穿了黑色学兰，也非常嫌弃小静你那手摸脏我衣服的啊。
　　──何况，搞清楚……不是你抓住我了。是我折原大爷特意大发慈悲地等在这个雨棚下让你捉住的。
　　──换作平常，你恐怕没有这等福利啊……小静。
　　
　　“你一个人在那边叨念什么？跳蚤。”静雄挑起半边眉，不耐烦地瞪着自己看。
　　“我说小静啊……”
　　“干么？别想求饶！不会放过你的。”
　　“求饶什么？”
　　
　　临也哼地笑了下，“没要你感谢我就很好了。”
　　
　　“啊？什么鬼，欠揍么。”静雄的额前爆出了一条青筋。
　　“咳咳。”
　　他拿腔拿调地清了清喉咙，说：“你不觉得，我既然都老实站在这儿等你了，接下来，你该有什么反应，懂我的意思？”
　　“……好好把握良机揍跳蚤一顿？”
　　“不是，你个草履虫筋肉脑啊……”他摀着额头为眼前的蠢蛋的低智商感到忧伤。“伞啊，我说的是伞！没有么？”
　　“伞？什么伞？”静雄愣了半晌才接过话道。
　　“现在可是在下大雨啊……你没有伞么？”
　　
　　对方仍抓住他的衣领不放，用另一只手挠了挠后脑梢，探头察看了棚外的雨势。
　　雨滴落得又重又急，打在塑料棚布上的雨声如此响亮，若是不打伞走在外头，肯定不出几秒，就会像掉进水坑一样淋个湿透吧。
　　平和岛静雄评估了一会儿，吶吶说道：“这种程度的雨，不就是手背遮一下，书包挡一下，赶快冲回家里就没事了么。何必打伞？”
　　
　　老天！他可真会被这个神经堪比四线道马路的家伙给打败。
　　
　　“小静，说这样不知体恤洗衣服的人辛苦的话，你妈妈会哭泣的噢。”
　　“烦死了。跳蚤，这么会说，那你倒是把伞借我啊。”
　　说完，挠着后脑的手顿时伸了过来，作势要抢自己提在手上的书包。
　　“哎，等一下，你干什么！”
　　“我说了把伞借我吧？”
　　“你那是拜托别人应有的态度么？”
　　“对跳蚤还需要用上客气的话语么？”
　　“居然用疑问句来反问我么！！”
　　
　　四只手交错拉扯着，就在书包的提把发出了快要被扯断的声音，情急之下，临也忍不住高呼了一声：“啊──真是的！我也没有啦！”
　　静雄停下了动作，“没有啥？”
　　“伞啦！伞！！”
　　临也别过头，从鼻子哼了口气，“还以为小静会带，高估你了……唔呼！”
　　他发出奇怪的声音，就因为牢骚话还没能说完，挟带空气的什么东西猛地从一旁袭了过来。本能地侧过头，定睛一看──
　　原来是对方朝自己正面挥来的拳头。
　　
　　“不敢置信，小静你居然是趁人家不注意就搞偷袭的阴险家伙么！太烂了！”
　　“谁烂啊！我刚才就说过要把握时间揍你了吧？还让你废话了一会儿才动手的好么？！”
　　　　
　　战火一触即发，两个人就在活动范围狭窄的雨棚下打起了架。而每当临也退到了逼近棚子的边缘，立刻就钻了空子，绕到静雄的背后去。
　　屡屡几次循环，再迟钝的人，都会发觉临也似乎在躲着什么。平和岛静雄开始想，依以前临也苗头不对立刻拔腿就跑的习性，今天倒显得反常啊。
　　而且，两个人这次保持的距离，比起前几次来说近得很多。
　　伸出手，彷佛就能马上碰触到对方似的。
　　察觉到这些事情的时候，已经接连举起几台脚踏车往临也那头扔去了。静雄稍微有点儿喘了。
　　
　　“喂……”
　　呼吸也变得有些急促的临也，手举高了刀对准静雄，稍带警戒地问：“干么？”
　　“你刚才问我伞的事，该不是要……”
　　他抢下对方的话，“事到如今才想要问我，我也无可奉告哦。”
　　“你该不是……不想淋雨吧。”
　　
　　临也沉默了。
　　并不想承认的事，被对方猜中了。
　　酸雨对头皮直至发根的伤害，眼前的暴力笨蛋是无法体会的。
　　那一头蓬松的金发，其根性恐怕如同本体的肌肉强度一样毫无道理地坚韧吧。
　　
　　──啊，果然一开始，就不该指望这家伙会带伞出门的。
　　
　　大概是看自己默不作声，将这份态度理解为“默认”。
　　从平和岛静雄的脸上，可以读到一系列的表情转变。由起初的疑惑，变为灵光一闪，最后像是美国卡通里的反派人物，露出了非常喜感的邪恶笑容。
　　
　　──什么啊，那种喜孜孜的表情。
　　
　　折原临也对此有着很不好的预感，可自己每次有了这种预感，不幸的是──有很大的概率都会应验。
　　大概起了什么坏点子，平和岛静雄扔掉手里有些变形的脚踏车，朝他的方向逼近了过来。
　　“嘿嘿，临也君哟，想不到你有这么可爱的弱点啊。”
　　临也不自觉向后退了两步。已经落入站在雨棚边缘，进退维谷的两难状态了。他的额前滑过冷汗，却硬假装成镇定自若的态度迎战对方。
　　“什么呀，笑得那样坏，小静是变态大叔么。”
　　
　　──不妙……小静笑得实在太异常地不怀好意了。
　　
　　无路可退的临也，眼角余光扫描过所有能够钻的空隙，可对方也顾及到了，防护得滴水不漏得周全。真该死，他想道。
　　“我只是想，既然你都这么表示了──”
　　话音未落，静雄一个箭步上前，把手构到临也后方，攫住他后颈的衣领。
　　“喂、你、想做什么！？”
　　临也扭着头试图挣脱对方的手，可一股蛮力却死拽着他不放。
　　“你都这么表示了，我不认真给点响应怎么行呢？”
　　静雄说完，没等临也意会那句话的意思，就被拖着出了雨棚外。豆大的雨点打在他们两人的手臂、头发、脸颊上。制服也因为一点一点吸收了雨水，颜色变得深暗。
　　“呜哇──住手，你这虐待狂，放开我！”
　　
　　临也从袖口甩出折刀拿在手里，迅速回身刺向静雄。
　　静雄以同样速度的条件反射，飞快松开手躲过攻击，随即又换了只手，再次拎住折原临也的衣领不放。
　　这种以身高差与手脚长短的明显落差彰显对方优势的作法，令屈居劣势的临也感到窝火。
　　简直瞧不起自己似的。
　　
　　在滂沱大雨中的激斗，泥泞被踩得四处飞溅，沾上鞋子、制服与裤子上。平常打架惯了，总是带着尘土污泥回家去的平和岛静雄丝毫不介意。制服能保持洁白也从来只在刚穿起走出家门的短暂片刻。可折原临也可就不同了。就算学兰是黑色的，长袖的制服饱吸雨水，附着了泥块而变得沉重，这都是相当恼人的触感。
　　可说是非常恶趣味的玩笑。
　　
　　混乱中，每当静雄稍微松开对临也的箝制，临也马上打算退回雨棚去，此时静雄又会追了过来，抓紧他的手腕不让跑出这片暴雨范围。
　　面对自己狼狈而又忿怒的表情，对手反倒露出相当痛快的脸。即使是在他的胳膊上划下一道道细红的伤痕，也被对方无关紧要地忽略了。
　　彷佛欣赏自己此刻的模样更是有趣似的。去死吧，这混账！平常不是该表现盛怒的样子么，此时为何好像立场颠倒了啊。临也不满地想道。
　　这场雨持续下了多久，两人间的架打了多久，完全搞不清楚。雨势过于猛烈，时间被凝聚在了一瞬间。
　　两人已被雨淋个湿透，平时张扬乱翘的发尾，也都服贴乖顺地淌着雨水顺脸颊而下。
　　
　　
　　“平和岛同学？还有折原同学？怎么搞得，这么晚了还在学校。”
　　略带责备语气的声音唐突打断了他们的缠斗。穿着雨衣雨裤，打着伞的学校值班老师拿起手电筒照了照两个人的脸。
　　“天色很暗了，怎么不回去？”
　　静雄打掉了临也准备趁他不备时朝心窝刺去的小刀，并抓住那不安份的手，“知道了，我们这就回去。”
　　老师疑惑地打量闪着光泽的金属碎片，“你们，不是在打架吧？”
　　“不是。”临也表现得就像个好学生一样得体，“我们在找掉在地上的钥匙。”
　　“找到了么？”
　　“找到了。”
　　“是么，那别逗留了，快回去。”
　　“是。”
　　
　　两人朝老师的方向弯身鞠了躬，草草拿上书包，转身就往校门口开溜。
　　照理说，出了校门就该各奔东西的，可小静不知哪根筋不对劲，死揪着自己的手不放，直往Sunshine 60大道奔去。
　　那与自己家的方向可是背道而驰的呀！
　　“小静，你这个笨蛋！快点放开我！”
　　在路上就这么大声嚷嚷了起来，好在身边的行人撑着伞低头赶路，谁也没注意到他们两个大男孩手牵着手跑在街道上。
　　可问题是，对方完全充耳不闻，仍执拗地紧抓着自己的手。黯淡的天色不时闪电交加，一明一灭；雷声轰鸣，周身人车鼎沸，接连不断的喇叭声音盖过了他的声音。
　　
　　太叫人生气了！连书包都湿了，里面的课本一定也泡了水，字迹都糊掉了。头发也因为沾到泥水变得又黏又硬……差劲，他真想一刀捅死眼前的混账。
　　前方的人没有停下，似乎想朝家的方向全力冲刺，卯足了劲地往前奔跑。被对方高大的个子挡住了视线，临也只能茫然无措地配合大怪兽的跑步节奏，跌跌撞撞在人群中穿梭，忍受人们的惊呼与嫌恶。
　　“停下来……快停下来！够了，讨厌……快点放开我！”
　　临也气恼地喊道。也不知道是不是终于成功传递给前面的人知道了，那个笨蛋突然停下来，搞得临也煞车不及，整个人撞在对方的手臂上。
　　胳臂又硬又突出，磕在鼻子上，疼得都要流泪了。打开嘴巴正想破口大骂。一阵喇叭声又预示了另一个大难临头── 一台货车急拐过路口，轮胎压过的水坑溅起人一样高度的水花，哗啦啦地淋在他们身上。
　　
　　“啊──眼睛……沙子跑进去了，混账……”临也不断揉弄着眼睛与鼻子。鼻水、眼泪、雨水，已经分不清是什么，他的脸上满是水痕，看上去一定特别愚蠢吧？
　　被抓紧的手突然松开来了，瞇起眼睛看出去的世界十分迷蒙，隐约似乎是看见了平和岛静雄放大的脸庞。
　　“怎么，跳蚤……你不是哭了吧？”
　　语气被非调侃，但也不像是带有关心之情。
　　“去死……你哪只眼睛看见我哭了！？”
　　“两只都是啊。”
　　“你应该去做视力检查了，草履虫。”

　　虽然眼睛还是难受得睁不开，眼角的余光却瞥见对方朝着自己把手伸了过来。他警戒地后退，掏出刀子指向对方：“不要靠近我，你这个怪物。”
　　那只伸过来的手在半空中僵住了，晃了一下，然后缩回去了。
　　他的刀子没法很好地瞄准对方，只不过是虚张声势罢了。纠缠下去对自己不利，见对方此时也没有动静，只是站在雨中望着自己，不趁此时开溜，更待何时？
　　“明天杀了你！”
　　扔下一句不知道算不算有气势的恫吓，临也扭过头拔腿就逃。
　　
　　他一边奋力在雨中狂奔，一边庆幸对方似乎没有追上来。
　　……Safe。　
　　他越跑越快，越跑越喘。直到推开家门，扑倒在玄关前才松了一口气。
　　觉得看见自家哥哥涨红了脸，呼哧呼哧喘着气的光景十分难得，两个拥有相同脸孔的妹妹凑了过来，围在他的身边东瞧西看。

　　“明明不准我们下雨时出去玩水，自己却玩泥巴玩得这么疯。临哥真狡猾！”
　　“ 兄，污。安？(哥哥弄得很脏，没事吧？)”
　　临也把手盖在脸上，不想搭理妹妹们的骚扰。
　　怎么说来，今天的事情都是场意外。就连失态也是。
　　比起疑惑平和岛静雄有别以往的异常行径，他更愤怒对方居然让自己在雨中淋个湿透这种事情。
　　
　　──该死的小静，居然那样嘲弄我。好样的，要是明早起床我的发量变少了，肯定不会让你接下來的一个月有安好的日子可过了。
　　
　　杀了他，一定要杀了他。那时的临也如此坚定地想道。可要是等到了明天，他从眼线那儿得来的情报，指出──呆愣在原地的平和岛静雄，遇上外出买东西的弟弟，并对他说了：“担心哥哥忘了带伞，早在出门前就把伞放进书包里面。”这样的事实……
　　届时，他又会作何想法，露出什么表情呢？


　　……爱的小伞？
　　……他与他么？


　　别开玩笑了。就算发生的机率不是绝对的零值，可光想象那种画面也足令他感到恶寒不断了。
