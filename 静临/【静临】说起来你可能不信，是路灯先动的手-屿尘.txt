# 

 
平和岛静雄觉得，自己今天心情简直糟透了。 
要说起收债和以前做过的那些工作有什么不同，静雄会告诉你两点： 
1.他的力气有了正大光明发挥的空间。 
2.从此告别正常工作日。 
是的，他目前的收债工作不分工作日和休息日，有任务的时候就一家一家跑，偶尔空起来一整天都无所事事。鉴于现在的人总是欠钱不还，静雄忙碌的日子会比较多，当然他完全不介意，因为就算放假，他也没什么可以打发时间的事情。 
不过放假总还是让人轻松的。 
例如今天。 
上一次社长给他放假好像还是因为祭典，这一次则是因为近期收债委托变少了。好几天前他就开始思考，难得的假日到底该做些什么好，原本以为自己会在家里宅上一天，结果还是出门了。 
今天是周三，路上没什么学生，大多数都是陌生的身影，他在池袋漫无目的地走动，期间目击工作中的赛尔提两次，被塞门强塞宣传单三次，遇到门田四人组一次，被狩泽问了奇怪的话…… 
诶，她问我什么了来着？ 
静雄挠挠头，将不好的东西选择性遗忘。 
总体来看，这个难得的假日过得平淡而普通，没什么特别让人高兴的事儿，不过也绝对谈不上糟糕。 
但是这一刻，静雄却认为，自己的心情一定差极了。 
要不然，他为什么会觉得眼前的路灯那么欠揍呢？！ 

 
# 

 
折原临也自诩是个冷静的人，尤其在遇到突发事件的时候。事实上，通常越是意料外的展开，越是能让他感受到乐趣。 
不过当他真正明白发生在自己身上的是什么样的事情后，他还是觉得这个玩笑开得有些过了。 
早上睁开眼睛的同时，他意识到天还没完全亮。 
那么问题就来了，理应睡在自家公寓大床上的他，为什么睁开眼睛就能看到天空和街道呢？ 
一瞬间，无数种可能性涌入了他的大脑。 
也许是有人在夜晚潜入了他的公寓，然后绑架了他。但这是不可能的，首先要潜入他公寓就是一个挑战，其次他睡眠很浅，一点风吹草动就能醒来，更何况被人绑到了外面。更重要的是，哪有人绑架别人绑到大街上的，不都是往地下扔么。 
灵魂出窍？ 
虽然是个无神论者，但目前的情况让临也没有选择的余地。 
尽管他感受不到自己的身体，视角却可以360度转动，他很快就辨认出这是池袋的一条小巷，他非常熟悉的地方。 
他想自己是漂浮在半空中的，目测眼睛到地面有两米八，并且贴着一边的墙壁，看样子是在第三根路灯和第五根路灯的正中间。 
嗯？ 
这一刻，他终于发现了。 
他成为了这条路上的第四根路灯。 

 
# 

 
平和岛静雄皱着眉头，抬起头盯着眼前的路灯。 
它看上去平淡无奇，黑色的灯柱掉了一小块漆，一看就是在这里竖了很多年的老路灯，每天都在特定的时间里默默为行人照亮眼前的道路，无论晴雨冷暖，它一直都散发着暖黄色的光。 
……画风好像不对了。 
总之，这样形象朴素的老路灯此刻在静雄眼中却笼罩在熟悉的恶劣气息之下。 
他皱着眉抬起头死死盯着面前的路灯，如果此时有人路过一定会觉得惊悚，池袋最强什么时候和路灯结下了深仇大恨？！ 
反正在临也眼中，静雄表现得像个小丑。他不明白为什么变成这样了静雄还能察觉到他的气息，但是看着对方对着一根路灯怒目而视的蠢样，临也觉得这次的灵魂出窍真是值回票价。 
而且能够以这样的视角居高临下地看着平和岛静雄，对他来说可是有生以来第一次。 
真是太有趣了，太有趣了，太有趣了！！！ 
虽然不能发出声音，但是这并不能影响临也的兴致，他在心里嚣张地大笑出声。 
然后紧接着，又发生了一件出乎意料的事情。 
在他笑的同时，路灯突然开始疯狂地闪烁。 

 
# 

 
平和岛静雄觉得，自己被一根路灯挑衅了。 
他原本就看这根路灯非常不爽，但是心里尚存的一丝理智告诉他，不可以和路灯较劲，不管长得多欠扁，它只是一根路灯，它是公共设施，随意损坏是要扣钱的，今天是难得的休假，他不能…… 
然后他就看到那破路灯肆无忌惮地闪烁起来，在下午四点天还未暗的时刻。这是赤裸裸地挑衅。 
他简直能想象某人在他面前挥舞着小刀一脸“你来揍我呀”的样子。 
这能忍吗？ 
静雄用他的拳头写下了答案：不能。 
而不能动的临也此刻开始觉得忧伤，这个游戏并不公平，他认为。不用看也知道自己所寄存的这根路灯已经断了，现在正在以肉眼可见的速度向地面靠近。不知道路灯有没有生命啊？如果就这么摔下去路灯不碎才怪，那么他呢？如果说他目前是以路灯为眼睛，那么碎了之后他会瞎吗？还是直接会死呢？ 
大概也只有折原临也会在这种时候考虑这样的问题。 
不过两种情况都没有发生，在砸到地面前的那一秒，他的世界突然天旋地转。 

 
# 

 
等回过神来的时候，临也发现自己还在半空中，距离地面两米八的地方。看来他的灵魂固定在了另一个路灯上，他依旧不能动，但他可以感受到来自身体的眩晕，如果可以的话他真想喝一杯水。 
临也开始怀念自己的身体了。坏消息是，他根本不知道回到自己身体的方法。 
另一边，收回拳头的一瞬间静雄就后悔了，再怎么暴躁也不能殴打路灯啊，他看着倒在地上已经完全失去作用的路灯，心中产生了忏悔之情。奇怪的是，现在他再看这个路灯，没有再觉得烦躁，也许刚才只是错觉。 
要冷静，要冷静。他不喜欢暴力。 
他快速穿过了小巷，这里离他的家并不远。静雄觉得自己需要去洗个澡，吃点东西，然后早点休息，用最好的面貌去迎接全新的明天—— 
可是就在他走出小巷的时候，路口的路灯毫无预兆地亮了起来。 

 
# 

 
在临也的脑中突然出现了一种毫无逻辑可言的可能性。 
他想要回到自己的身体中，不过目前看来这件事情凭他自己是无法完成的，他甚至没有可以求助于外人的方法，那么他能够仰仗的，大概就只剩下平和岛静雄了。 
他现在已经掌握的情况是：他的灵魂会随机固定在任何一根路灯上，而只有在路灯遭到损坏的时候，才会出现灵魂转移的情况。 
那么只要不断地损坏路灯，他也许就有机会回到自己的身体里。尽管这样的机会可能会出现在全池袋的路灯都倒下之后。 
但这是唯一的解决方法了。他要做的事情也很简单，那就是不断地挑衅静雄，不断地引导对方，在这一点上，世界上不会有第二个人比折原临也更加擅长。专注给静雄找事儿七年，假一赔十。 
所以当他看到静雄走出小巷的时候，他让自己亮了起来，并且以一种奇怪的频率快速闪烁着。 

 
# 

 
静雄喘着气。 
当眼前的路灯亮起来的时候，他真的试图冷静。但是就像他看到折原临也就控制不住自己想要追过去打一样，他看到那路灯就忍不住一拳揍上去。 
他怀疑临也是不是在池袋的每一个路灯内都装了摄像头，他总觉得对方正透过灯泡盯着自己，这让他觉得非常不舒服。 
而且每当他收拾完一个路灯的时候，不远处一定会有另一个路灯亮起来，不远不近，正好保持在他的视线范围之内，仿佛生怕他不注意到一样，还一定会不停地闪烁来强调自己的存在感。 
这么恶劣的把戏，一定是临也那小子搞的鬼没跑了。 
原本静雄还打算今天放过临也一马，不过现在他改变主意了，正好那些路灯向新宿延伸过去，他干脆一路杀气腾腾地向新宿冲去。 
他心里那点破坏公共财产产生的罪恶感已经被怒火消磨得一干二净。 
说出来你可能不信，但这次真的是路灯先动的手。 

 
# 

 
临也觉得这一切都是命运石之门的选择。 
就算闪灯这件事情他已经炉火纯青，但是他确实没有办法控制自己下一次会转移到什么地方去。不过就像是冥冥之中自有安排，他发现这个路线的终点似乎就是他所住的高级公寓，当然事情总是喜忧参半的，他看着怒气值爆表的平和岛静雄，觉得等一下在自家公寓里一场恶战是难以避免了。 
当然，如果让事情真的就这么顺利发展下去，平和岛静雄也不会成为折原临也最讨厌的人没有之一。 
这里是临也公寓楼下，是最后的路灯了。临也心中有一种感觉，下一次他就会成功回到自己的身体里，他闪得比任何时候都勤奋，看着平和岛静雄整个儿黑掉的脸，在心里默念快来揍我吧快来揍我吧。 
可是这一次静雄并没有一拳上来，而是将路灯直接拔了出来。 
临也一脸惊恐地看着自己被静雄提了起来，抗在肩上。 
卧槽！小静这是要干什么！他怎么不按常理出牌啊！！！！！！ 
如果静雄是打算用路灯胖揍临也，那么折原临也将有幸成为有史以来第一个目睹“自己”揍自己的人类。 
当静雄站在他公寓门口的时候，临也已经心如死灰。 
而接下来，静雄突然用手中的路灯，狠狠地朝门砸去。 

 
# 

 
临也睁开眼睛的同时，听到自家大门传来的哀嚎。 
不难想象那扇门已经呈现一种怎样的报废的状态，如果他不快些行动的话，那么下一个报废的可能就是他自己。可是，该死的，他的头好晕，灵魂不停移动的后遗症让他特别想吐，他想要就这么睡死在自己的床上。 
可是楼梯上传来的脚步声就像是死亡倒计时。 
他身上还穿着睡袍，口袋里空空如也，他艰难地从枕头底下摸出了备用的小刀，强忍着呕吐的欲望从床上坐了起来，同时，平和岛静雄一脚踹开了他卧室的门。 
“不管小静是来找我做什么的，现在我没空哦。”如果可以的话临也一点都不想打架，怎么看都会输得很惨。他要尽自己所能和平地将静雄赶走。 
“那些路灯是你搞的鬼吧？！” 
好难受好难受好难受。想死。 
临也放弃般地任由自己无力的身体再次瘫回到床上，他闭上眼睛，觉得口干舌燥。 
“就算你不信，不过这次真的不是我啊……我还没这么无聊在路灯上装摄像头……呃……” 
说出来才惊觉大事不妙，这和此地无银三百两有什么区别！ 
他感到静雄爬上了自己的床，此刻双手撑在自己头边，他处于绝对的劣势，这样的认知让他浑身僵硬。 
“果然是你小子做的啊，”静雄的声音在耳边响起，听上去低沉又沙哑，“你一直透过那些路灯在看着我，对吧。” 
他的呼吸打在临也的脖子边。 
“不如，今晚让你看个够好了。你说怎么样，临——也——老——弟——哟……”
